Щоб додати кнопку входу за допомогою Google на свій вебсайт, потрібно використовувати OAuth 2.0 для автентифікації користувачів через Google. Ось основні кроки та приклад коду:

Зареєструйте свій проект в Google API Console:

Перейдіть на Google API Console.
Створіть новий проект або виберіть існуючий.
Увімкніть Google+ API для вашого проекту.
Перейдіть до "Credentials" і створіть OAuth 2.0 Client ID.

Налаштування OAuth 2.0 Client ID:

Вкажіть дозволені URI перенаправлення (наприклад, http://localhost:8000/oauth2callback для локального розробки).

Додайте бібліотеку Google API до вашого проекту:

Додайте до вашого проекту необхідну бібліотеку JavaScript для Google API.

Напишіть HTML код з кнопкою входу за допомогою Google:

ПЕРШИЙ ФАЙЛ

Обробка на сервері:
На сервері потрібно перевірити токен, надісланий від клієнта. Приклад для Node.js з використанням бібліотеки google-auth-library

ДРУГИЙ ФАЙЛ

В прикладі вище, CLIENT_ID замініть на ваш реальний Client ID, отриманий в Google API Console.

Запустіть сервер і тестуйте:

Запустіть свій сервер та відкрийте HTML сторінку в браузері.
Переконайтесь, що вхід через Google працює та відправляє правильний токен на сервер.

Це базовий приклад. Для реального застосування вам може знадобитись додаткова обробка помилок, зберігання сесій користувача та інші функції.
